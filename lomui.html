          # Welcome to LomUI

          # 快速上手

          1.您可以在Github上下载LomUI的最新版本 

          https://github.com/Minit-ChengWangying/LomUI/tree/master

          2.在您的代码中引入他 
          <!DOCTYPE html>
<head>
    <title>Document</title>
    <link rel="stylesheet" href="css/minit.css">
    <link rel="stylesheet" href="css/Popup.css">
</head>
<body>

    <script type="text/javascript" src="js/jquery.min.js"></script>
    <script type="text/javascript" src="js/minit.js"></script>
</body>
</html>

          完成以上两步您即可使用LomUI

          # 页面布局

          ！！！

          LomUI在version1.0和version2.0版本中，会自动对页面进行迷小标准化主题适应，如果您不希望我们这样，可以使用LomUI version3.0版本以上。

          # 表单

          输入

          <input type="text" id="minit_input">

          按钮

          <button id="minit_btn"></button>

          Tips

          如果您不希望使用我们提供的点击动效，您可以将此id去掉

          # 弹出层

          如果要使用弹出层，您需要引用以下代码

          <link rel="stylesheet" href="css/Popup.css">
<script type="text/javascript" src="js/Popup.js"></script>

          在您的代码中添加一个DIV

          <div class="pop_container" id="popup">
    <p class="popText">Hello</p>
</div>

          您即可使用JavaScript调用弹出层

          Popup("Hello");

          警告！

          使用PopUp();函数必须传递参数，否则会报错
